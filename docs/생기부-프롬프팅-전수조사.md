#### 나이스 바이트 계산 규칙
```python
def count_bytes(text):
    total = 0
    for char in text:
        if '\uAC00' <= char <= '\uD7A3':  # 한글 음절 → 3B
            total += 3
        elif '\u3131' <= char <= '\u318E':  # 한글 자모 → 3B
            total += 3
        elif char == '\n':  # 줄바꿈 → 2B
            total += 2
        else:  # 영어/숫자/특수문자/공백 → 1B
            total += 1
    return total
```

1. **전문성 있는 자기주도 연구자**: 과학·기술 기초와 응용 지식을 스스로 탐구·확장

4. **배려·협력(Caring)**: 높은 윤리의식과 협업으로 사회 공헌

#### 서울대 인재상 5역량
1. **도전·혁신**: 진취적 태도·회복탄력성
2. **문제해결**: 비판적 사고 + 디지털 문해
3. **의사소통**: 공감·협업·다양성 존중
4. **사회공헌**: 시민의식·사회적 책임감
5. **글로벌·융합 리더**: 초학제적 시야·국제 협업

#### 서울대 종합역량평가 3대 축
1. **종합사고역량**: 깊이 있는 지식 → 분석·비판 사고·판단력
2. **창의탐구역량**: 호기심 → 지식 연계·확장 → 스스로 탐구 방법 수립
3. **공동체역량**: 타인 존중, 긍정적 가치관·윤리 의식 실천

#### 창의탐구역량 요구 행동 6가지
1. 주도적 과제 추진 (문제 정의→설계→실행→수정·심화)
2. 지식 확장·연계 (교과→다른 교과/실생활 연결·융합)
3. 도전적 탐구 (실패 후 변수 조정, 재도전 흔적)
4. 창의적 문제 해결 (새 방법/형태/심미적 감각)
5. 사고의 유연성 (꼬리물기 질문, 탐침 질문)
6. 의사소통·협력 (팀 조율, 외부 협력, 설득)

#### 과학고 전략
- KAIST: 심화 탐구를 **스스로 설계** → 데이터 분석·모델링·실험 → 실질적 해결책/프로토타입
